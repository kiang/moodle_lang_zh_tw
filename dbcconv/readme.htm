深藍UTF-8正體簡體轉換函數 1.0

  1. 引用方法

    • 將壓縮檔解壓縮到您喜歡的位置
    • 在在欲使用的程式中，require , require_once, include , include_once 將 dbcconv.php 引入，例如 include("dbcconv/dbcconv.php");


  2. 使用方式

    • string dbcconv( string text [, int encode]) : 提供 UTF8 編碼之中文正體與簡體互轉 (第一參數為欲轉換之文字；第二參數為輸出之字體，0 表正體、1 表簡體，預設為 1 )
      範例 : echo dbcconv("深藍UTF-8正體簡體轉換函數", 1);
      輸出 : 深蓝UTF-8正体简体转换函数
